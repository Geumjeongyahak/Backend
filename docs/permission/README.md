# Permission Docs

현재 `docs/permission` 문서는 권한 모델 설계와 API 접근 정책을 함께 정리하는 공간입니다.

- `현재 접근 정책`: 코드에 반영된 `@PreAuthorize`, `/me` 예외 경로, 역할 기반 접근 제어를 기록합니다.
- `permission code 초안`: 아직 설계 중인 권한 코드를 도메인별로 정리합니다.
- `표 기준`: `권한/접근 조건 | API 목록` 형태로 요약합니다.
