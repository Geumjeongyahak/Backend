## 문서 기준

- 공통 응답 구조와 문서 구조는 [API 안내 문서](../api-spec/api_spec.md)를 기준으로 봅니다.
- 에러 코드 의미와 체계는 [Error Codes](../error_codes.md)를 기준으로 봅니다.
- 이 폴더의 문서는 엔드포인트 레퍼런스뿐 아니라 권한, side effect, 대표 실패 케이스, 시퀀스 다이어그램까지 함께 다룹니다.
