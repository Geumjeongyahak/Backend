# 손모음 플랫폼 PRD (Product Requirements Document)

## 1. 개요

### 1.1 프로젝트 배경

금정열린배움터는 부산대학교 인근 지역 기반 교육 봉사기관으로, 어르신 문해교육을 통해 교육 사각지대를 해소하고 있습니다.

**현재 문제점:**
- 출결, 일정, 결강, 재정 관리가 수작업으로 이루어져 업무 효율이 낮음
- 강의 교환 절차가 복잡함
- 학습 진도 관리 미흡으로 맞춤형 피드백 제공이 어려움
- 공식 소통 창구 부재로 공지 전달이 비효율적임

### 1.2 목표

1. **업무 자동화** - 출결/일정/결강/재정 관리 디지털화
2. **재정 투명성 강화** - 결제 이력 및 입출금 기록 DB 기반 관리
3. **학습 모니터링 지원** - 개별 학습자 진도 실시간 확인
4. **커뮤니케이션 활성화** - 반별/부서별 게시판 지원

## 2. 사용자 정의

### 2.1 사용자 유형

| 역할 | 설명 |
|------|------|
| **VOLUNTEER (봉사자)** | 수업을 진행하는 교사 |
| **ADMIN (관리자)** | 시스템 및 운영 관리자 |
| **STUDENT (학생)** | 수업을 받는 학습자 (선택사항) |

### 2.2 역할별 주요 기능

#### 봉사자 (VOLUNTEER)
- 출석 체크 및 이력 조회
- 수업 일정 조회 (담당 분반/과목)
- 수업 일지 작성/수정
- 결석 요청 및 수업 교환 요청
- 기자재 구입 요청

#### 관리자 (ADMIN)
- 사용자/권한 관리
- 분반/부서/과목 관리
- 요청 승인/반려 (결석, 교환, 구입)
- 전체 현황 대시보드
- 데이터 내보내기

#### 학생 (STUDENT) - 선택사항
- 본인 출석 확인
- 공지 확인
- 일정 확인
