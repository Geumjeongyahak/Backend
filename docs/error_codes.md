# Error Codes

## 1. 개요

금정야학 API는 공통 에러 코드와 도메인 비즈니스 에러 코드를 분리합니다.

- 공통/인증/인가/시스템 에러는 도메인 번호 없이 관리합니다.
- 도메인에 밀접한 비즈니스 에러만 `PREFIX-도메인번호-일련번호` 형식을 사용합니다.
- 같은 도메인 안에서는 `RES`, `BIZ`, `VAL` 접두사별로 일련번호가 독립적으로 증가합니다.

## 2. 코드 형식

### 2.1 공통 코드

| Prefix | 설명 | 예시 |
|---|---|---|
| `AUTH` | 인증 실패 | `AUTH001` |
| `AUTHZ` | 인가 실패 | `AUTHZ001` |
| `VAL` | 공통 입력 검증 오류 | `VAL001` |
| `BIZ` | 공통 비즈니스 상태 오류 | `BIZ004` |
| `SYS` | 서버/외부 시스템 오류 | `SYS001` |

### 2.2 도메인 코드

형식: `PREFIX-도메인번호-일련번호`

| Prefix | 의미 |
|---|---|
| `RES` | 리소스를 찾을 수 없음 |
| `BIZ` | 도메인 비즈니스 충돌/상태 오류 |
| `VAL` | 도메인 입력값/규칙 검증 오류 |

도메인 번호는 `src/test/resources/sql/init_scheme.sql`의 테이블 생성 순서를 기준으로 할당합니다.

| 도메인 번호 | 도메인 |
|---|---|
| `01` | Users |
| `02` | Departments |
| `03` | Classrooms |
| `04` | Students |
| `05` | Subjects |
| `06` | Lessons |
| `07` | Requests |
| `08` | Channels |
| `09` | Posts |
| `10` | Comments |

## 3. 공통 에러 코드

| 코드 | HTTP Status | 설명 |
|---|---|---|
| `RES001` | 404 | 요청한 리소스를 찾을 수 없습니다. |
| `VAL001` | 400 | 입력값 검증에 실패했습니다. |
| `VAL002` | 400 | 잘못된 입력값입니다. |
| `VAL003` | 400 | 필수 필드가 누락되었습니다. |
| `VAL004` | 400 | 변경된 값이 없습니다. |
| `VAL007` | 405 | 지원하지 않는 HTTP 메서드입니다. |
| `BIZ001` | 409 | 이미 존재하는 리소스입니다. |
| `BIZ004` | 409 | 유효하지 않은 상태입니다. |
| `SYS001` | 500 | 서버 내부 오류가 발생했습니다. |
| `SYS002` | 500 | 데이터베이스 오류가 발생했습니다. |
| `SYS003` | 500 | 외부 API 호출 중 오류가 발생했습니다. |
| `SYS004` | 500 | 파일 업로드에 실패했습니다. |
